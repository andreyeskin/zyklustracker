StatistikManager

open class StatistikManager

StatistikManager - Zentrale Logik für alle Statistik-Berechnungen Diese Klasse ist verantwortlich für: - Laden und Filtern von Daten aus der Datenbank - Berechnung aller Statistiken (Zyklus, Stimmung, Schmerz, Symptome) - Asynchrone Verarbeitung im Background Thread - Callback-basierte Kommunikation mit der UI Alle Datenbankzugriffe laufen in Background Threads!
