AnalyseErgebnis

open class AnalyseErgebnis

AnalyseErgebnis - Datenmodell für Zyklusanalyse-Ergebnisse Diese Klasse kapselt die Ergebnisse der Zyklusanalyse und Sensor-Bewertung. Sie wird von ZyklusPhaseBerechnung verwendet, um strukturierte Rückmeldungen über den aktuellen Zyklusstatus und die Bewertung der Sensor-Werte zu geben. Design-Pattern: Value Object für unveränderliche Analyse-Ergebnisse
